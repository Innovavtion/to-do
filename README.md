# General information

Make a project with the **standard** functionality of the to-do list, as well as add **additional** functionality.

**Stack: Yarn, React, SaSS/SCSS**

## Objective of the project

- Learn to use **Yarn**
- Learn to use **React**
- Learn to use **SaSS/SCSS**
- Learn to use **ESLint**
- Learn to use **Prettier**
